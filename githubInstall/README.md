# Middle School Coding Setup Check

## Assignment Goal
Today you are proving that you can set up the coding tools we use in class **yourself** and make them work.

This is **not** a beginner "Hello World" lesson. Most of the software may already be installed on your computer. Your job is to **check that everything is installed correctly, fix anything missing, and prove that Python works in VS Code and that GitHub Desktop is connected and usable**.

---

## Learning Targets
By the end of this assignment, I can:

- install or verify **GitHub Desktop**
- install or verify **Visual Studio Code**
- install or verify **Python**
- open a project folder in **VS Code**
- make sure **Python runs correctly in VS Code**
- use **GitHub Desktop** with a repository

---

## What You Must Check
You need to confirm that these tools work on your computer:

- **GitHub Desktop**
- **Visual Studio Code**
- **Python 3**

If one is missing or not working, your job is to fix it.

---

## Part 1: Check GitHub Desktop
1. Open **GitHub Desktop**
2. Make sure you are signed in to your GitHub account
3. If GitHub Desktop is not installed, install it
4. Once it opens, take a screenshot or show your teacher that:
   - GitHub Desktop opens correctly
   - you are signed in
   - you can see a repository or create one

### GitHub Desktop Check
Complete these tasks:
- [ ] GitHub Desktop opens
- [ ] I am signed in
- [ ] I can create or clone a repository

---

## Part 2: Check Visual Studio Code
1. Open **Visual Studio Code**
2. If VS Code is not installed, install it
3. Make sure it opens correctly
4. Open a folder for today’s work

### VS Code Check
- [ ] VS Code opens
- [ ] I opened a project folder

---

## Part 3: Check Python
1. In VS Code, create a new file called:

```python
python_setup_check.py
```

2. Type the following code:

```python
import sys

print("Python is working.")
print("Python version:", sys.version)
print("Success! My coding setup is ready.")
```

3. Save the file
4. Run the file in VS Code

### Python Check
Your program should display:
- a message saying Python is working
- your Python version
- a success message

### Python Check List
- [ ] Python is installed
- [ ] My Python file runs in VS Code
- [ ] I can see the output in the terminal

---

## Part 4: Fix Any Problems
If something does not work, troubleshoot it.

Possible issues:
- GitHub Desktop is not installed
- VS Code is not installed
- Python is not installed
- Python is installed but not selected in VS Code
- The file will not run because it was not saved correctly

If you get stuck, write down:
- what went wrong
- what you tried
- what fixed it

---

## Part 5: Repository Check
Once everything works:

1. Create a repository called:

```text
coding-setup-check
```

**OR** clone the class repository if your teacher tells you to.

2. Add your `python_setup_check.py` file to the repository folder
3. Make sure GitHub Desktop can see the file change

### Repository Check List
- [ ] I created or cloned a repository
- [ ] My Python file is inside the repository folder
- [ ] GitHub Desktop shows my changes

---
